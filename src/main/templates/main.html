Welcome to the API Page!

To use our API, follow these steps:

  1. Open your application or tool for sending HTTP requests.
  2. Make a GET request to the following address: http://161.35.221.15/api/
  3. Specify the name parameter in the request to get information about a specific cryptocurrency. For example: http://161.35.221.15/api/?name=bitcoin
  4. Expect a JSON response with information about the cryptocurrency.

Example response:

        {
            "name": "Bitcoin",
            "price": 46000,
            "last_updated": "2022-01-12T12:34:56Z"
        }
